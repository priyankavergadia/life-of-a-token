# Project: Tools — For Everyone (no coding needed)

How an AI becomes a "digital worker" that **uses tools** and **chains steps** to
answer a real business question — instead of guessing.

👉 **Open it:** https://priyankavergadia.github.io/life-of-a-token/#/tools

---

## Before you start
Add a key in the **⚡ Go live** bar (any provider). No key → demo trace.

---

## Walk it
1. **Overview** — Question → Pick a tool → Run it → Chain + answer.
2. **Forge tools** — Meet two tools: one looks up sales revenue, one calculates
   profit margin. *Lesson: the AI reads each tool's description to decide when to
   use it; math is done by code, so numbers are never made up.*
3. **Execute** — Edit the executive's question and click **Run the agent**. Watch
   the trace: it fetches revenue **first**, then feeds it into the margin
   calculator **second** — no human in between. *Lesson: the AI plans and chains
   the steps itself.*

## ✅ What you'll understand
The leap from a chatbot that *talks* to an agent that *acts* — and why giving it
tools makes it reliable on things LLMs are bad at (math, live data).
