# Greeting Cards — For Everyone (no coding needed)

The final project: a team of **five AI agents** that work together — like a design
studio — to create a hand-drawn greeting card, with a self-correcting **critique
loop**.

👉 **Open it:** https://priyankavergadia.github.io/life-of-a-token/#/greeting-cards

---

## Walk the steps
1. **Overview** — read how five agents form a pipeline: brainstorm → fetch art →
   lay out → critique → render.
2. **The Team** — meet each agent (each is just a *name + a job*):
   - **Connie** writes the text and picks 6 doodles · **Archie** fetches the art ·
     **Layla** arranges it · **Chris** critiques it · **Ren** renders the final card.
3. **Critique Loop** — the key idea: **Chris** checks the design and, if something’s
   wrong (off-theme items, too much text, overlap), sends specific feedback back to
   whoever caused it. The team fixes it and tries again — up to 6 times.
4. **The Result** — see the finished card and how it’s given a hand-drawn look.

## ✅ What you'll understand
How multiple AI agents + a critique loop turn a request like *“a birthday card for
a 1-year-old”* into a finished, polished product — the core idea behind “agentic” AI.
