# Project: Personalization — For Everyone (no coding needed)

The finale: combine **meaning-based matching**, **creative writing**, and **strict
formatting** to generate a personalized marketing email at scale.

👉 **Open it:** https://priyankavergadia.github.io/life-of-a-token/#/personalization

---

## Before you start
Add a key in the **⚡ Go live** bar. The matching step needs **Gemini** or
**OpenAI** (Claude has no embeddings). No key → demo mode.

---

## Walk it
1. **Overview** — Match (by meaning) → Create (creative copy) → Lock to JSON →
   Auto-send.
2. **Matchmaker** — Edit what the customer bought and click **Match**. Related
   products (e.g. boots, rain jacket for a tent) rise to the top; unrelated ones
   stay away. *Lesson: recommendations by meaning, not keywords.*
3. **Generate** — Click **Generate**. See the same result two ways: a friendly
   marketing email *and* the strict JSON behind it (subject, body, code, button).
   *Lesson: you can be creative AND machine-readable at the same time.*

## ✅ What you'll understand
How real systems get **creative output they can still trust** — copy that's fun to
read but locked into a format an email service can actually send.
